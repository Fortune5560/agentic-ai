# Agentic AI

Agentic AI is an education-first platform focused on how autonomous AI agents are designed,
deployed, and operated in real-world systems.

## What This Site Covers
- Agentic system blueprints
- Tool comparisons for AI workflows
- Human-in-the-loop (HITL) design
- AI governance and ethical practices

## Philosophy
Agentic AI prioritizes:
- Safety and auditability
- Machine-readable content
- Practical, production-ready systems

This repository hosts the static website for Agentic AI.
